I've set some specific goals:

Alright, that might sound a bit high-flown. But now, for some important organizational matters we'd like answers to. We know we're working in the terminal, and we want to have at least a two-player mode. It would be great if you could play against the computer.

Hey, classic tic-tac-toe is fun, but what if we go further? So, let's add another requirement. Local multiplayer. A player vs. the computer? - Of course. A player, their buddy, and the computer on the same board? - Challenge accepted.

The idea from the beginning was this: we want to use a concept like server and client. The board handles the game's logic. The player is a semi-abstract interface that, in short, allows us to interact with the board. Well, later it turns out that being a semi-interface is equivalent to a "players" list.

Our Very Own AI!

                        import random
                        
                        class Board:
                            def __init__(self,changed_size=3):
                                self.size = changed_size
                                self.board = [[' ' for _ in range(self.size)] for _ in range(self.size)]
                        
                                self.game_ended = False
                                self.moves = 0
                                self.column_indicator = '|'
                                self.row_indicator = '_'
                        
                        
                            def get_size(self):
                                return self.size
                            
                        
                            def get_board(self):
                                return self.board
                            
                        
                            def get_game_ended(self):
                                return self.game_ended
                        
                        
                            def show(self):
                                for row in self.board:
                                    print(f" {self.column_indicator} ".join(row))
                                    if row != self.board[-1]:
                                        print(self.row_indicator * self.size**2)
                        
                                print('\n')
                        
                        
                            def make_guess(self,input):
                                x,y = input[0]
                                symbol = input[1]
                                if x > self.size or y> self.size:
                                    print(f"Please, use values (0,{self.size - 1})") 
                                    return False
                                if self.board[x][y] != ' ':
                                    print("This place is already taken") 
                                    return False
                                self.board[x][y] = symbol
                                
                        
                                if self.check_winner(symbol):
                                    self.game_ended = True
                                    print(f'\nCongrats, {symbol} wins!!!\n')
                                    return True
                        
                                self.moves += 1
                        
                                if self.checkDraw():
                                    self.game_ended = True
                                    print(f"\nIt's a draw\n")
                                    return True
                        
                                return True
                        
                        
                            def is_valid_coordinate(self,x,y):
                                return 0 <= x < self.size and 0 <= y < self.size
                                
                        
                            def is_valid_move(self,x,y):
                                return self.is_valid_coordinate(x,y) and self.board[x][y] == ' '
                            
                        
                        
                            def check_winner(self, symbol):
                                # Check rows
                                for row in self.board:
                                    if all(mark == symbol for mark in row):
                                        return True
                        
                                # Check columns
                                for col in range(self.size):
                                    if all(self.board[row][col] == symbol for row in range(self.size)):
                                        return True
                        
                                # Check diagonals
                                if all(self.board[i][i] == symbol for i in range(self.size)) or \
                                all(self.board[i][self.size - i - 1] == symbol for i in range(self.size)):
                                    return True
                        
                                return False
                            
                            def checkDraw(self):
                                '''
                                Check there is no chance for anyone to win.
                                '''
                                return sum(row.count(' ') for row in self.board) < self.size
                        
                        
                        class Player:
                            def __init__(self, symbol):
                                self.symbol = symbol
                        
                            def makeMove(self,game_board: Board):
                                x,y = -1,-1
                        
                                while not game_board.is_valid_move(x,y):
                                    while True:
                                        try:
                                            # x,y = input("Your coordinates in format row col :").split()
                                            x, y = map(int, input("Your coordinates in format row col: ").split())
                                            if x < 0 or x >= game_board.get_size() or y < 0 or y >= game_board.get_size():
                                                print(f"Coordinates must be between 0 and {game_board.get_size() - 1}.")
                                            else:
                                                break
                                            
                                        except ValueError:
                                            print("Please enter a valid coordinates")
                        
                                
                                x,y = int(x),int(y)
                                return (x,y),self.symbol
                                
                                
                        class PlayerAI:
                            def __init__(self, symbol,difficulty='easy',board_size=3):

                                self.difficulty = difficulty

                            def makeMove(self,game_board: Board):
                                x,y = 1,1
                                while not game_board.is_valid_move(x,y):
                                    x,y = random.randint(0,self.size -1),random.randint(0,self.size -1)
                                
                                # print(f"x:{x},y:{y} \n")
                                return (x,y),self.symbol
